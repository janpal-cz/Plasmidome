# Plasmidome of chicken gut
This GitHub repository accompanies a manuscript https://doi.org/10.1128/spectrum.04074-25. 

It provides Python scripts for plasmidome-associated analyses, including normalization of resistance, plasmid and MGE content abundance, and visualization of the data using Matplotlib. 
Data tables and images generated by these scripts are included in this repository, see `Results/`.

# Full manuscript citation
Rysava M, Stredanska K, Schwarzerova J, Jakubickova M, Cejkova D, Aytan-Aktug D, Otani S, Dolejska M, Palkovicova J. Dynamic changes in the plasmidome and resistome in the gastrointestinal tract of chickens. _Microbiol Spectr_ 0:e04074-25, https://doi.org/10.1128/spectrum.04074-25.
